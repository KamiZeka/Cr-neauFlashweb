Pour publier sur GitHub Pages :

1. Dézippez ce fichier.
2. Envoyez tout le contenu à la racine du dépôt GitHub Pages : index.html, CNAME, presentation-creneauflash.pdf et le dossier assets.
3. Attendez que GitHub redéploie le site.
4. Ouvrez https://creneauflash.fr

Ne touchez pas aux DNS mail/MX sur IONOS.
